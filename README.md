# Introduction to Web Applications (2025/2026)

Materials for the _[Introduction to Web Applications](https://github.com/polito-iwa-2026)_ course, offered to third-year students of the Bachelor's Degree in Computer Engineering at Politecnico di Torino.

## Repository Structure

The course materials are organized into the following folders:

- **📚 exercises/**: Contains the exercises and examples covered during lectures.
- **💻 labs/**: Each lab has its own folder with the lab description and its solution (published one week after the lab session).
- **🎓 slides/**: Contains the presentations used during the course.
